# Viltrum Card

Premium Sepolia testnet crypto card website.

## Features

- Telegram login flow
- RainbowKit wallet connect
- Virtual card purchase
- Physical card purchase
- Free mint
- Supabase database
- Admin approval
- Telegram bot card viewer
- Physical card shipment tracking

## Deployment

Frontend: Vercel  
Database: Supabase  
Bot: Render/Railway  
Contracts: Sepolia testnet
